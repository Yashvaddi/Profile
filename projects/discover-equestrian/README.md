# Discover Equestrian

Discover Equestrian is a platform for users to **list properties, businesses, and services for sale or rent**, with paid listings, powerful search, favorites, and chat between buyers/renters and sellers. 

- Live URL: `https://discover-equestrian.com`
- Role: Frontend Engineer (Next.js/React)

## Tech stack

- Next.js / React.js 
- TypeScript
- Styling: Tailwind CSS / Bootstrap / Material UI
- State Management: React Query / Redux / Context API
- Realtime: Socket.IO (if chat is real-time)
- Testing: Jest
- Integrations: Payments, Google Maps (if used) 

## Key features

- Paid property and services listing creation and management. 
- Search and filtering to quickly find relevant listings. 
- Favorite/shortlist properties for quick access.
- Chat with sellers to streamline property negotiations and closing. 

## Architecture & micro frontends

- Feature-based folder structure (listings, search, chat, profile).
- Shared UI components reused across multiple modules for consistency.
- Components designed to be independently deployable in a micro frontend setup.
