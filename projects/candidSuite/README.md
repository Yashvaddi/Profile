# CandidSuite - Applicant Tracking System

CandidSuite is a comprehensive **Applicant Tracking System (ATS)** that streamlines and automates the hiring process, providing detailed reports and analytics for data-driven recruiting decisions. 

- Live URL: `https://candidsuite.com`
- Role: Frontend Engineer (Next.js/React, analytics dashboards)

## Tech stack

- Next.js / React.js 
- TypeScript
- Styling: Tailwind CSS / Material UI
- State Management: React Query / Redux
- Testing: Jest
- Charts/Analytics: (e.g., Recharts / Chart.js) if used
- Others: API integration, role-based access

## Key features

- Candidate management: applications, profiles, and pipelines. 
- Recruitment workflows: stages, status tracking, assignments.
- Reports and analytics dashboards for hiring performance. 
- Responsive, recruiter-friendly UI with filters and search.

## Architecture

- Feature-sliced modules for candidates, jobs, pipelines, and analytics.
- Reusable components for tables, filters, and cards.
- Test-friendly code with container/presentation separation where applicable.
