# Nuway Roofing

Nuway Roofing is a platform with a **dynamic UI for quoting, project management, and DIY tools**, focused on performance, responsiveness, and real-time updates for a seamless user experience. 

- Live URL: `https://nuway-roofing.demoserver.biz`
- Role: Frontend Engineer (React/Next.js UI)

## Tech stack

- React.js / Next.js 
- TypeScript
- Styling: Tailwind CSS / Bootstrap
- State Management: React Query / Redux / Context API
- Realtime: Socket.IO (if real-time updates)
- Testing: Jest

## Key features

- Interactive quote generation flows for roofing projects. 
- Project management views for tracking jobs and statuses. 
- DIY tools for users to estimate or plan their own projects.
- Highly responsive, optimized UI for both desktop and mobile. 

## Architecture

- Modular feature folders for quotes, projects, DIY tools.
- Shared UI primitives for forms, cards, and layout.
- Micro frontend-friendly separation between main app sections.
