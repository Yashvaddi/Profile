# Sales CRM

Sales CRM is a full-featured platform for **managing leads and opportunities** with multi-channel communication (call, SMS, email, WhatsApp, video call) and modules for project tracking and lead requirement management. 

- Role: Frontend Engineer (Next.js/React, real-time communication)

## Tech stack

- Next.js / React.js 
- TypeScript
- Styling: Tailwind CSS / Bootstrap / Material UI
- State Management: React Query / Redux / Context API
- Realtime & Communication: Socket.IO, WebRTC (for video calls) 
- Testing: Jest
- Integrations: Telephony/SMS/Email APIs, WhatsApp API (as applicable)

## Key features

- Lead and opportunity management with pipeline stages. 
- Multi-channel communication: calls, SMS, email, WhatsApp, video. 
- Project tracking and lead requirement modules. 
- Dashboards for sales performance and activity.

## Architecture

- Separation between communication modules and CRM core.
- Micro frontend-ready modules (e.g., communication widget vs CRM views).
- Strong focus on UI state management and testable interfaces.
