# BoatDox

BoatDox is a web platform that helps yacht brokers **create, execute, and securely store listing transaction documents** with end-to-end workflows, dashboards, and guided data collection for smooth listings and sales. 

- Live URL: `https://boatdox.org`
- Role: Frontend Engineer (Next.js/React, micro frontend-friendly UI)

## Tech stack

- Next.js / React.js 
- TypeScript
- Styling: Tailwind CSS / Bootstrap (choose what you used)
- State Management: React Query / Redux / Context API
- Testing: Jest
- Others: API Integration, AWS (S3/CloudFront), Docker/Nginx if applicable 

## Key features

- Secure document creation, execution, and storage for yacht listings.
- Role-based dashboards for brokers, admins, and other stakeholders. 
- Guided flows to collect required listing information and complete paperwork.
- Responsive, performant UI optimized for daily broker workflows. 

## Micro frontend & architecture

- Modular UI components organized by domain (listing, documents, dashboard).
- Clear separation between shared components and feature-specific modules, making it micro frontend-ready.
- Testable components with props-driven logic and Jest coverage for core flows.
